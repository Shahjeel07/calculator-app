
Neon Glass Calculator
=====================

This is a single-file HTML calculator app (index.html) with embedded CSS and JavaScript.
Open `index.html` in any modern browser (desktop or mobile) to run it.

Features:
- Attractive neon glassmorphism design
- Keyboard support and touch support
- Ripple animation and theme toggle
- Copy result by swiping left (mobile)

Provided as a ZIP for easy download.
